🚁 Flight Operations Center Dashboard

Prototype Presentation — Front-End Web Development Demo for Aviation & Emergency Response Systems

Description

Front-end prototype per il coordinamento di missioni aeree e antincendio, con interfaccia di mission control integrata.

Key Features

  • Mission Management: aggiunta, modifica e tracking (Active, Standby, Resolved).
  • Geospatial Visualization: mappa interattiva (Leaflet.js) con marker personalizzati (🔥, 🚁, 🟠, 🟢).
  • Weather Integration: struttura pronta per dati meteo utili a pianificazione e risk assessment.
  • Crew & Aircraft Assignment: quick-pick per assegnare elicotteri ed equipaggi.
  • Operational Timeline: orari di inizio/fine missione per scheduling e allocazione risorse.
  • UI/UX Focus: interfaccia chiara, pilot-friendly, in vanilla HTML/CSS/JS.

Technologies

HTML · CSS · JavaScript

⛅ Live Weather App

Prototype Presentation — Front-End Web Development Demo for Weather & Geolocation

Description

App meteo leggera e responsive con UI accessibile e geolocalizzazione.

Key Features

  • City Search & Geolocation: ricerca con suggerimenti + geolocalizzazione nativa.
  • Live Conditions: temperatura, feels like, vento, umidità, precipitazioni, qualità aria (US AQI).
  • 5-Day Forecast: card giornaliere con min/max, vento, alba/tramonto, precipitazioni.
  • Hourly Trend: sparkline delle prossime 24 ore.
  • Dynamic Background: gradienti animati in base al meteo.
  • UX Utilities: toggle unità (°C/°F, km/h/kt), tema light/dark con persistenza.
  • Accessibility & Performance: ruoli ARIA, contrasti ottimizzati, vanilla stack.

Technologies

HTML · CSS · JavaScript · Open-Meteo API
